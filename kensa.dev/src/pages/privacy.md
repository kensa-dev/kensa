# Privacy

This site uses Google Analytics (GA4) to understand how visitors use the documentation — pages visited, time on page, and approximate location (country/region). IP addresses are anonymised before being sent to Google.

No personal data is collected, sold, or shared with third parties.

If you prefer not to be tracked, you can opt out using [Google's opt-out browser add-on](https://tools.google.com/dlpage/gaoptout).
